# Testing Checklist

## Conversion

- Convert 1 hectare to square meters.
- Convert 1 carreau to square meters.
- Convert decimal values using both period and comma input.
- Swap source and destination units.
- Copy a result.

## Costing and currency

- Enter land cost in USD, HTG, and H$.
- Change the result currency.
- Test a manually entered USD-to-HTG rate.
- Confirm H$1 equals HTG 5.
- Test behavior when the live-rate request fails.

## Interface

- Switch between English and French.
- Open and close each accordion.
- Reload and confirm remembered panel state.
- Test light and dark device appearance.
- Test a narrow mobile viewport.
- Navigate controls with a keyboard.

## PWA and offline

- Load once while online.
- Install to a home screen when available.
- Reopen while offline.
- Confirm the application shell and conversions work.
- Confirm cached exchange-rate behavior is clearly reported.
